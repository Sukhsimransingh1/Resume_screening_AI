# Resume Screening AI

A machine learning web application that analyzes resumes against job descriptions using BERT-based semantic similarity, skill matching, and experience evaluation.

Live Demo: https://sukhsimran-singh-resume-screening-ai.hf.space

## Features
- Resume and job description matching
- Section-wise scoring (BERT, skills, experience)
- Skill gap identification (matched and missing skills)
- AI-based improvement suggestions
- PDF resume parsing
- Interactive split-dashboard UI
- Deployed using Hugging Face Spaces

## Model Details
- Model: BERT-based (multi-qa-mpnet-base-dot-v1)
- Task: Semantic similarity between resume and job description
- Framework: PyTorch + Sentence Transformers
